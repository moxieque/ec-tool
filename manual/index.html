Moxieque 専用 / 非エンジニア向け / 2026年4月版

1. はじめに

このツールは、ラクマートを通じた仕入れ・在庫・販売の管理を一元化するためのものです。Webブラウザから操作できるため、特別な知識は不要です。

このツールでできること

📊
ダッシュボード
売上・仕入れ・粗利をひと目で確認。期間選択でトレンドを把握。
📥
仕入れ管理
ラクマートの配送依頼書を自動取込。国際送料の按分計算も自動。
📤
販売登録
販売情報を入力すると、原価・粗利がリアルタイムで表示される。
📦
在庫一覧
在庫数を直接編集。少ない商品には警告アイコンが表示される。
💰
利益レポート
月次・商品別・チャネル別に利益を分析。
🏷️
商品マスタ
商品情報の登録・編集・CSV一括管理。
⚙️
設定
アラート閾値や手数料率の設定、バックアップ操作。

動作環境

項目要件
OSmacOS（推奨）
ブラウザGoogle Chrome / Safari / Firefox（最新版）
Python3.10 以上
インターネット接続ラクマート自動取込時のみ必要
ℹ️ このツールについて
このツールはあなたのPC上でのみ動作します。データはすべてローカルに保存されており、外部サーバーには送信されません。

2. インストール・起動方法

macOS での起動

  1. ツールフォルダを開く
    Finderでツールフォルダを開きます。
  2. 「Start.command」をダブルクリック
    黒いウィンドウ（ターミナル）が開き、自動でセットアップ・起動が始まります。
  3. ブラウザが自動で開く
    数秒待つとブラウザでツールが自動的に表示されます。
ℹ️ セキュリティ警告が出た場合
macOSのセキュリティ設定により「開発元を確認できない」と表示される場合があります。その場合は：
Start.command を右クリック → 開く → 開く

Windows での起動

  1. ツールフォルダを開く
    エクスプローラーでツールフォルダを開きます。
  2. 「Start.bat」をダブルクリック
    黒いコマンドプロンプトが開き、自動でセットアップ・起動が始まります。
  3. ブラウザが自動で開く
    数秒待つとブラウザでツールが自動的に表示されます。
✅ これだけでOK！
「Start.command」（Mac）または「Start.bat」（Windows）をダブルクリックするだけで、ライブラリのインストールからブラウザを開くまですべて自動で行われます。初回は数分かかる場合があります。
⚠️ ターミナル/コマンドプロンプトは閉じないでください
自動で開いたウィンドウはツールが動いている間は開いたままにしてください。閉じるとツールが停止します。

初回のみ：Python が入っていない場合

「Start.command」または「Start.bat」を実行してインストーラーが自動で開いた場合は、以下の手順でインストールを完了させてください。

ℹ️ インストーラーは自動でダウンロード・起動されます
Python が入っていない場合、スクリプトが公式サイトから自動でダウンロードしてインストーラーを開きます。サイトを探したりダウンロードボタンを押す必要はありません。

macOS の場合

  1. インストーラーが自動で開く
    黒いウィンドウにメッセージが表示されインストーラーが起動します。
  2. 「Continue」→「Continue」→「Agree」→「Install」の順に進む
    途中でMacのパスワード入力を求められます。PCにログインするときのパスワードを入力してください。
  3. 「Installation Successful」と表示されたらウィンドウを閉じる
  4. 黒いウィンドウに戻り、Enterキーを押す
    自動でツールの起動を続けます。

Windows の場合

  1. インストーラーが自動で開く
    黒いウィンドウにメッセージが表示されインストーラーが起動します。
  2. インストーラーの画面で「Add Python to PATH」にチェックを入れる
    左下の「Customize Installation」をクリックして進めます。
  3. 「Install Now」をクリック
    インストールが進行します。完了するまで待ってください。
  4. 「Setup was successful」と表示されたらウィンドウを閉じる
  5. 黒いコマンドプロンプトに戻り、任意のキーを押す
    自動でツールの起動を続けます。

ツールの終了方法

  1. ブラウザのタブを閉じる
  2. ターミナル/コマンドプロンプトのウィンドウを閉じる

3. ダッシュボードの見方

ツールを開くと最初に表示されるのがダッシュボードです。事業のKPI（重要指標）をひと目で確認できます。

画面の構成

表示項目説明
売上合計選択した期間の総販売金額
仕入れ合計選択した期間の総仕入れ金額（国際送料含む）
粗利売上 − 仕入れ原価 − 手数料
粗利率粗利 ÷ 売上 × 100（%）
販売件数選択した期間の販売登録数

期間の切り替え方

  1. 画面右上の期間選択プルダウンをクリックする
  2. 表示したい期間を選ぶ：
    • 今月：今月のデータのみ
    • 3ヶ月：直近3ヶ月分
    • 6ヶ月：直近6ヶ月分
    • 1年：直近12ヶ月分
    • 全期間：登録済みの全データ
  3. 数値・グラフが自動で更新される

グラフの見方

グラフには月ごとの売上・粗利の推移が表示されます。棒グラフの高さで規模感を把握し、線グラフで粗利のトレンドを確認してください。

ℹ️ ヒント
グラフの棒や線にマウスを乗せると、その月の詳細な数値がポップアップで表示されます。

4. 仕入れ管理

仕入れの登録には3つの方法があります。ラクマートを使っている場合は「自動取込」が最も便利です。

A. ラクマート自動取込（推奨）

ラクマートの配送依頼書からデータを自動で読み込みます。

  1. 「📥 仕入れ管理」タブをクリック
  2. 「ラクマート取込」エリアを確認
    画面上部に「ラクマートサーバー: 稼働中 ✅」と表示されていればOKです。
    「停止中」の場合は「起動」ボタンを押してください。
  3. 配送依頼書一覧を取得する
    「配送依頼書一覧を取得」ボタンをクリックします。過去の依頼書が一覧で表示されます。
  4. 取り込む依頼書を選択
    各行の左端のチェックボックスにチェックを入れます（複数選択可）。
  5. 「🚀 一括取得」ボタンをクリック
    選択した依頼書の商品情報が自動取得されます。
    ※ 依頼書1件あたり20〜30秒かかります。複数選択時はその分時間がかかります。
  6. 按分フォームで金額を確認・調整
    国際送料などの追加コストが自動で各商品に按分されます。必要に応じて金額を修正してください。
  7. 「登録」ボタンで仕入れを確定
⚠️ 注意：初回ログインが必要
初めてラクマート取込を使う際は、ブラウザでラクマートにログインする必要があります。詳しくは「10. ラクマート連携設定」を参照してください。

B. 個別登録

1件ずつ手動で仕入れを入力する方法です。

  1. 「📥 仕入れ管理」タブで「個別登録」セクションを開く
  2. フォームに以下を入力する：
    項目説明
    商品名プルダウンから登録済み商品を選択
    仕入れ数量仕入れた個数
    単価（元）ラクマートでの商品単価（人民元）
    国際送料（円）この商品にかかる送料の按分額
    仕入れ日実際の仕入れ日
    仕入れ先ラクマート等のサプライヤー名
  3. 「登録」ボタンをクリック

C. CSV一括取込

Excelなどで作成したCSVファイルから複数の仕入れデータをまとめて登録できます。

  1. 「CSVアップロード」ボタンをクリック
  2. CSVファイルを選択する（文字コード: UTF-8）
  3. プレビューで内容を確認して「登録」をクリック
ℹ️ CSV形式について
CSVのフォーマットは「8. 商品マスタ」のCSVダウンロード機能でサンプルを取得できます。列の順序を変えないでください。

按分計算の仕組み

国際送料は複数商品でまとめて支払われることが多いため、商品ごとの原価計算に「按分」が必要です。このツールでは仕入れ数量で均等に按分します。

✅ 例
A商品を5個、B商品を5個仕入れ、国際送料が1,000円だった場合：
→ A商品：1,000円 ÷ 10個 × 5個 = 500円
→ B商品：1,000円 ÷ 10個 × 5個 = 500円

5. 販売登録

商品を販売したら、このタブで記録を登録します。原価・粗利はリアルタイムで自動計算されます。

販売を登録する

  1. 「📤 販売登録」タブをクリック
  2. フォームに必要事項を入力する：
    項目説明
    商品名 必須プルダウンから選択。選択すると原価が自動表示される
    販売数量 必須販売した個数
    販売単価 必須1個あたりの販売価格（円）
    販売チャネルAmazon / メルカリ / Yahoo! / その他
    手数料率販売チャネルの手数料（%）。設定のデフォルト値が自動入力
    販売日実際の販売日
    メモ任意のメモ
  3. 粗利を確認
    フォーム下部に「原価」「販売小計」「手数料」「粗利」が自動計算されます。
  4. 「登録」ボタンをクリック
ℹ️ 原価の計算方法
原価は移動平均法で自動計算されます。仕入れのたびに単価が更新されるため、常に最新の平均原価が使われます。

登録済み販売データを編集・削除する

  1. 「販売履歴」テーブルで対象の行を探す
  2. 行右端の「✏️ 編集」ボタンをクリックするとフォームに内容が読み込まれる
  3. 内容を修正して「更新」ボタンをクリック
⚠️ 削除は元に戻せません
「🗑️ 削除」ボタンをクリックすると確認ダイアログが表示されます。削除後は復元できないため、バックアップを取ってから行ってください。

6. 在庫一覧

現在の在庫数をリアルタイムで確認・編集できます。

在庫一覧の見方

列説明
商品名 / SKU商品マスタに登録された名前と管理コード
現在庫数現在の在庫数量（直接編集可）
移動平均原価直近の仕入れデータから自動計算された1個あたりの原価
在庫評価額在庫数 × 移動平均原価
状態⚠️ アイコンが表示されている場合は在庫が少ない（要補充）

在庫数を直接編集する

  1. 対象商品の「在庫数」セルをクリック
  2. 新しい数値を入力してEnterキーを押す（または別のセルをクリック）
  3. 変更が自動保存され、変更履歴が記録される
ℹ️ 在庫数が自動で変わるタイミング
  • 仕入れ登録時：在庫数が増加します
  • 販売登録時：在庫数が減少します
  • 直接編集時：入力した数値に変更されます（棚卸し時などに使用）

在庫アラートの設定

在庫が指定数以下になると ⚠️ マークが表示されます。閾値は「⚙️ 設定タブ」で変更できます（初期値: 3個）。

7. 利益レポート

販売・仕入れデータを集計して、様々な切り口で利益を確認できます。

レポートの種類

レポート内容活用シーン
月次レポート 月ごとの売上・原価・粗利・粗利率の一覧 月次振り返り、季節性の把握
商品別レポート 商品ごとの売上・粗利・粗利率の一覧 売れ筋・利益商品の特定
チャネル別レポート Amazon・メルカリ等の販売チャネルごとの集計 出品先の比較・最適化

レポートの読み方

  • 粗利率が高い商品＝利益効率が良い商品。仕入れを増やす候補。
  • 売上が高いが粗利率が低い商品＝売れているが利益が薄い。価格見直しや仕入れコスト削減を検討。
  • チャネル別粗利率＝手数料の違いが影響します。手数料の低いチャネルで売れると利益が改善。
ℹ️ 粗利の計算式
粗利 = 販売金額 − （原価 × 数量） − 手数料
手数料は販売登録時に入力した手数料率で計算されます。

8. 商品マスタ

商品の基本情報を管理します。仕入れ・販売登録で使う商品はここに登録が必要です。

商品を1件登録する

  1. 「🏷️ 商品マスタ」タブをクリック
  2. 「商品追加」フォームに以下を入力する：
    項目説明
    商品名 必須管理しやすい名前を入力
    SKU 必須自社管理コード（例: MOX-001）。重複不可
    ラクマートURL商品ページのURL（画像自動取得に使用）
    メモサイズ・カラーなど補足情報
  3. 「登録」ボタンをクリック

CSV で一括ダウンロード

  1. 「CSVダウンロード」ボタンをクリック
  2. 商品マスタ全件がCSVファイルとしてダウンロードされます
  3. ExcelやGoogleスプレッドシートで開いて内容を確認・編集できます
⚠️ CSVの文字コード
ダウンロードされるCSVは UTF-8 BOM付き 形式です。Excelで開くと文字化けなく表示されます。アップロード時も同じ形式を使用してください。

CSV から一括アップロード

  1. ダウンロードしたCSVをExcelで編集して保存する
  2. 「CSVアップロード」ボタンをクリックしてファイルを選択
  3. プレビューで内容を確認してから「インポート」をクリック
⚠️ SKUの変更に注意
SKUは仕入れ・販売・在庫データと紐づく重要なキーです。後から変更すると、過去のデータとの整合性が崩れる可能性があります。SKUは最初に正しく設定してください。

商品を削除する

商品マスタの各行右端の「削除」ボタンで削除できます。ただし、仕入れや販売データが存在する商品は削除しないでください（データの不整合が生じます）。

10. ラクマート連携設定

ラクマートの配送依頼書を自動取込するには、初回に一度ブラウザでログインが必要です。

初回ログイン手順

⚠️ 事前確認
この操作はツール（サーバー）が起動している状態で行ってください。起動方法は「2. インストール・起動方法」を参照。
  1. ツールを起動してブラウザで開く
    http://localhost:8080 にアクセスします。
  2. 「仕入れ管理」タブを開く
    ラクマートサーバーが「稼働中」になっていることを確認します。「停止中」の場合は「起動」ボタンを押します。
  3. 「配送依頼書一覧を取得」をクリックする
    初回は自動的にブラウザが開き、ラクマートのログイン画面が表示されます。
  4. ラクマートにログインする
    自動で開いたブラウザ（Chromium）でラクマートのIDとパスワードを入力してログインします。
  5. ログイン完了後、ブラウザを閉じずに待つ
    ツールが自動でデータを取得します。しばらく待つと配送依頼書の一覧が表示されます。
ℹ️ 2回目以降
ログイン情報はPC上のブラウザに保持されます。次回からは自動でログインされるため、手動ログインは不要です（ただし、ログアウトした場合や長期間経過した場合は再ログインが必要です）。

ラクマートサーバーについて

ラクマート連携には rakumart_server.py（ポート8766）というプログラムが使われます。これは app.py を起動した際に自動で起動されます。

項目内容
ポート番号8766
起動タイミングpython3 app.py 実行時に自動起動
確認方法設定タブ または 仕入れ管理タブのサーバー状態表示

11. バックアップ・復元

データはすべてPC内のJSONファイルに保存されています。定期的なバックアップを強くお勧めします。

データの保存場所

moxieque-ec-tool/
└── data/
    ├── products.json        商品マスタ
    ├── purchases.json       仕入れデータ
    ├── sales.json           販売データ
    ├── inventory_history.json 在庫編集履歴
    └── config.json          設定

バックアップを取る方法（ツールから）

  1. 「⚙️ 設定」タブをクリック
  2. 「バックアップをダウンロード」ボタンをクリック
  3. ZIPファイルがダウンロードされます。定期的に外付けHDDやクラウドストレージに保存してください。

バックアップを取る方法（手動）

  1. Finderで moxieque-ec-tool/data/ フォルダを開く
  2. フォルダ全体をコピーして、別の場所（デスクトップ、外付けHDDなど）に貼り付け
  3. フォルダ名に日付を付けて保存（例：data_backup_20260402）
⚠️ 商品画像もバックアップする
商品画像は static/images/ フォルダに保存されています。別PCへ移行する際や万一の場合に備えて、このフォルダもバックアップしてください。

データを復元する方法

  1. ツールを停止する（ターミナルで Ctrl + C）
  2. 現在の data/ フォルダを別の場所に退避（念のため残しておく）
  3. バックアップした data/ フォルダを元の場所に貼り付け
  4. ツールを再起動する（python3 app.py）
⚠️ 現時点の制限
設定タブの「バックアップから復元」ボタンは現在準備中です。復元は上記の手動手順で行ってください。

推奨バックアップ頻度

頻度タイミング
週1回以上毎週末など定期的に
大量データ入力後仕入れや販売を大量に登録した直後
PC移行・OSアップデート前作業前に必ずバックアップを取る

12. トラブルシューティング

よくある問題と解決方法

🔴 ブラウザで「このサイトに接続できません」と表示される

原因
サーバー（app.py）が起動していません。
  1. ターミナルを開く

🔴 ラクマートサーバーが「停止中」のまま起動しない

  1. 「⚙️ 設定」タブ → 「ラクマートサーバー起動」ボタンをクリック
  2. それでも起動しない場合は、ツール自体を再起動する（ターミナルで Ctrl+C → python3 app.py）

🔴 ラクマート取込が「タイムアウト」する

原因と対処
ラクマートのサーバーへのアクセスに時間がかかっています。1件あたり20〜30秒かかります。複数件を選択した場合はさらに時間がかかります。ローディング表示が出ていれば正常に処理中です。しばらく待ってください。

🔴 「ラクマートにログインしてください」と表示される

  1. 自動で開くChromiumブラウザでラクマートにログインする
  2. 「10. ラクマート連携設定」の手順を参照

🔴 在庫数がマイナスになっている

原因
販売登録した数が、仕入れ登録した数を超えています。仕入れの登録漏れがないか確認してください。
  1. 仕入れ管理タブで該当商品の仕入れ記録を確認
  2. 漏れがあれば仕入れを追加登録
  3. それでも解決しない場合は在庫一覧から直接修正する

🔴 商品マスタをCSVでアップロードしたが文字化けする

  1. CSVファイルをUTF-8（BOM付き）で保存し直す
  2. Excelの場合：「名前を付けて保存」→ ファイル形式で「CSV UTF-8（コンマ区切り）」を選択
  3. 再度アップロードを試みる

🔴 ツールが急に遅くなった / 動かなくなった

  1. ターミナルで Ctrl + C を押してサーバーを停止する
  2. ブラウザのキャッシュをクリアする（Cmd + Shift + R）
  3. python3 app.py でサーバーを再起動する

🔴 PCを買い替えた / 別のPCで使いたい

  1. 旧PCの moxieque-ec-tool/ フォルダ全体をコピー（外付けHDDまたはクラウドストレージ経由）
  2. 新PCに同じ場所（またはわかりやすい場所）に貼り付け
  3. 新PCでPython・ライブラリをインストール（「2. インストール・起動方法」参照）
  4. python3 app.py で起動する
ℹ️ 解決しない場合
上記で解決しない場合は、ターミナルに表示されているエラーメッセージをスクリーンショットまたはコピーして、開発者にお問い合わせください。
